
## 项目结构
```
doctorsAdvice -> vue的his医嘱项目

Drugstore_HZWeb -> 药店针对B端的商城项目(华忠医药)

    Drugstore_HZWeb
        |--drugstore_pc(药店针对B端的商城PC端)
        |--drugstore_server_web(药店针对B端的商城管理后台)

Drugstore_Web -> 药店针对C端的商城项目(金诃药店连锁)

    Drugstore_Web
        |--drugstore_pc(药店针对C端的商城PC端)
        |--drugstore_server_web(药店针对C端的商城管理后台)
        |--drugstore_wap(药店针对C端的商城移动web端)

emrAndHis    -> VUE的his系统的电子病历

JXCloud_APP  -> 九信云混合APP

JXCloud_web  -> 远程医疗管理系统

JY_Cloud-app -> 九悦云混合APP

JY_Web_DEV   -> 九悦云影响平台

OAWeb        -> 企业内部关系APP

```

## vue的his医嘱项目结构

```
    src
      |--assets(静态资源文件)
      |--com(项目整体页面包括组件)
         |--command(弹窗，表格公共组件)
         |--model(页面公共js文件)
         |--modules(页面组件)
         |--store(vuex)
         |--view(首页及登录页面)
      |--libs(项目公共js)
         |--cordovaAPI.js(api)
         |--upload.js(处理服务器路径js)
         |--util.js(工具类js)
      |--router(路由信息)
      |--styles(部分默认公共样式)
      |--App.vue
      |--main.js

```
